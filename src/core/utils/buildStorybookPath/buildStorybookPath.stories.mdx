# buildStorybookPath

Use this function to build a path for a Storybook doc.

## Code

```typescript
import { buildStorybookPath, StorybookSection } from '@core/utils';

const docPathForMyAwesomeComponent = buildStorybookPath(
    StorybookSection.COMPONENTS,
    'MyAwesomeComponent'
);

console.log(docPathForMyAwesomeComponent);
```
