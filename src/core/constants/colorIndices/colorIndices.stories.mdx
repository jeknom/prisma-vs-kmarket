# colorIndices

Contains constant values for the color indices that are used when accessing Mantine color theme values.

```typescript
import { colorIndices } from '@core/constants';
import { createStyles } from '@mantine/core';

const useStyles = createStyles((theme) => ({
  element: {
    backgroundColor: theme.colors.dark[colorIndices.PRIMARY_BACKGROUND_COLOR_INDEX]
  },
}));
```
